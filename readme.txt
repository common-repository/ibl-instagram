Instagram Photo Gallery is display Customizable Instagram Images/Videos Gallery with Lightbox effect.


== Description ==
Instagram Photo Gallery plugin allow users to display Instagram Gallery with lots of customization options with different layouts including Grid, Carousel. You can display Images/Video from Your Instagram account using shortcode. This plugin is very easy to use and completely responsive. 

Using **[ibl-instagram-gallery id=123]** shortcode, you can publish gallery in any Page or Post in your WordPress sites.

== Installation ==
1.	Install the IBL Instagram Gallery plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the /wp-content/plugins/ directory).
2.	Activate the plugin through the Plugins menu in WordPress admin.

3.	Add new gallery with configure your access token from Instagram Photo Gallery menu in WordPress admin.

4.	Get your Access token from here Get Access Token by [here](https://iblinfotech.com/instagram-photo-gallery-wordpress-plugin-for-instagram-feeds/instagram-generate-token/)

5.	Use the shortcode [ibl-instagram-gallery] with id in your page or post to display your Instagram images/videos.

== Frequently Asked Questions ==

= How many photos can I display in my Instagram feed? =

Moreover, in the pro version of the Instagram feed have bestest option to select.In this case, user is allowed to enter number for displaying the instagram feed 

= How to increase the speed of Instagram image loading? =

The loading time depends on the number of Instagram images in the Instagram grid. You can display less Instagram images to speed up the loading.

= What is the purpose of Instagram Access Token? = 

For using Instagram plugins and displaying Instagram photos right on a website you have to go with Instagram API. Access Token is exactly what you need for such operation.Instagram Access Token is key which allows getting instagram images and  additional information about them from Instagram, such as likes counter, comments and author name. 

= Does this plugin work with all WordPress versions? =

This version works with WordPress version upto 5.5.1.
